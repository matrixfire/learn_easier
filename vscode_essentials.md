# VS Code Python 后端程序员必备功能

## 核心功能

### 代码导航
- **符号跳转** (`F12`) - 直接跳转到函数/类定义
- **查找引用** (`Shift+F12`) - 查看所有使用某函数的地方
- **大纲视图** - 快速浏览文件结构

**CS 原理**：基于 AST（抽象语法树）分析，编译器理论的基础

### 智能代码补全
- **IntelliSense** - 基于上下文的代码建议
- **参数提示** - 函数参数实时提示

**软件工程**：IDE 通过类型系统和依赖分析提供帮助，减少认知负荷

### 重构功能
- **重命名符号** (`F2`) - 自动重命名所有引用
- **提取方法** - 将代码片段提取为独立函数
- **提取变量** - 将表达式赋值给变量

**CS 原理**：静态分析和语义分析，确保程序正确性

### 调试功能
- **断点** (`F9`) - 暂停执行
- **单步执行** (`F10`/`F11`) - 逐行或逐函数调试
- **条件断点** - 满足条件才暂停

**软件工程**：调试器原理，程序状态检查和跟踪

## 常用快捷键

### 文件操作
- `Ctrl+P` - 快速打开文件（模糊搜索）
- `Ctrl+Shift+P` - 命令面板
- `Ctrl+B` - 切换侧边栏

**CS 原理**：高效算法（模糊字符串匹配），时间复杂度优化

### 编辑操作
- `Alt+Up/Down` - 移动行
- `Ctrl+D` - 选择下一个相同单词
- `Ctrl+Shift+K` - 删除整行

**软件工程**：文本编辑器的高效操作，减少重复劳动

### 面板操作
- `Ctrl+`` (反引号) - 打开终端
- `Ctrl+Shift+E` - 资源管理器
- `Ctrl+Shift+G` - 全局搜索

**软件工程**：集成开发环境的理念，工具链整合

### 多光标编辑
- `Alt+Click` - 添加多光标
- `Ctrl+Alt+Up/Down` - 多光标向上/下扩展

**CS 原理**：并行编辑思想，同时处理多个位置

## 底层 CS 知识

### 语言服务器协议 (LSP)
- IDE 与语言工具的标准化通信
- 支持跨编辑器的一致体验

**CS 原理**：客户端-服务器架构，协议设计

### 抽象语法树 (AST)
- 代码的结构化表示
- 实现代码分析和转换的基础

**CS 原理**：编译器前端理论，语法分析

### 类型系统
- 静态类型检查
- 接口推断和验证

**CS 原理**：类型理论，程序正确性证明

### 进程管理
- 终端进程控制
- 后台任务执行

**CS 原理**：操作系统进程管理，IPC（进程间通信）

### 调试协议 (DAP)
- 调试器与编辑器的标准化通信
- 支持多种语言和调试器

**CS 原理**：调试器架构，运行时分析
