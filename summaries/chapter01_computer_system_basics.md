# 第01章 计算机系统基础知识

## 章节概述

**考试分值**: 2-6分（单项选择题）
**内容范围**: 计算机硬件基础、软件基础、计算机语言、多媒体技术

---

## 1. 计算机系统概述

### 定义与组成
- **计算机系统**: 硬件、软件及网络组成的系统
- **硬件**: 机械、电子元器件、磁介质和光介质等物理实体
- **软件**: 按特定顺序组织的数据和指令，控制硬件完成功能

---

## 2. 计算机硬件基础

### 冯·诺依曼结构
五大组成部分：
1. 运算器
2. 控制器（与运算器集成 → CPU）
3. 存储器
4. 输入设备
5. 输出设备

### 专用处理器
| 类型 | 用途 | 特点 |
|------|------|------|
| GPU | 图形处理 | 数百/数千内核，并行计算 |
| DSP | 数字信号处理 | 实时处理，哈佛体系结构 |
| FPGA | 现场可编程 | 可编程逻辑门阵列 |

### 指令集系统
- **CISC**: Intel、AMD x86（复杂指令集）
- **RISC**: ARM、Power（精简指令集）
- **国产处理器**: 龙芯、飞腾、申威（RISC-V、MIPS、ARM）

### 存储器层次
访问速度（高→低） | 容量（小→大）
- 片上缓存
- 片外缓存
- 主存（内存）
- 外存

### 总线
| 类型 | 特点 | 应用 |
|------|------|------|
| 并行总线 | 多条双向数据线，适合近距离 | 系统总线 |
| 串行总线 | 一条双向数据线，适合长距离 | 通信总线 |

## 3. 计算机软件基础

### 软件分类
- **系统软件**: 不依赖特定应用，管理软硬件资源
- **应用软件**: 解决特定问题，与具体领域相关

### 操作系统分类
| 类型 | 特点 |
|------|------|
| 批处理系统 | 作业=程序+数据+作业说明书 |
| 分时系统 | 多路性、独立性、交互性、及时性 |
| 实时系统 | 快速响应、高可靠性 |
| 网络系统 | 共享网络资源 |
| 分布式系统 | 透明性、可靠性、高性能 |
| 嵌入式系统 | 微型化、可定制、易移植 |

### 数据库类型
- 关系型数据库
- 键值（Key-Value）数据库
- 列存储数据库
- 文档数据库

### 文件系统
**文件类型**: 系统文件、库文件、用户文件 / 临时文件、档案文件、永久文件 / 只读文件、读写文件、可执行文件

**存储空间管理**（磁盘分配表）:
- 空闲区表
- 位示图（0/1表示空闲/占用）
- 空闲块链

### 中间件分类
1. 通信处理中间件（MQSeries）
2. 事务处理中间件（Tuxedo）
3. 数据存取管理中间件
4. Web服务器中间件（Tomcat、JBOSS）
5. 安全中间件
6. 跨平台中间件
7. 专用平台中间件
8. 网络中间件

### 软件构件标准
| 标准 | 规范 | 主要内容 |
|------|------|----------|
| CORBA | OMG | ORB、公共对象服务、公共设施 |
| J2EE | SUN | EJB（会话Bean、实体Bean、消息驱动Bean） |
| DNA 2000 | Microsoft | DCOM/COM/COM+ |

## 4. 计算机语言

### 语言层次
1. **机器语言**: 计算机直接执行（操作码+操作数）
2. **汇编语言**: 符号化描述（名字+操作符+操作数+注释）
3. **高级语言**: C、C++、Java、Python
4. **建模语言**: UML

### UML基本要素

#### 四种事物
- **结构事物**: 类、接口、协作、用例、主动类、构件、制品、节点
- **行为事物**: 交互、状态机、活动
- **分组事物**: 包
- **注释事物**: 注释

#### 四种关系
| 关系 | 含义 |
|------|------|
| 依赖 | 使用关系，独立事物变化影响依赖事物 |
| 关联 | 拥有关系，特例：聚合、组合 |
| 泛化 | 特殊/一般关系（继承） |
| 实现 | 接口与类、用例与协作 |

**聚合 vs 组合**:
- 聚合: 部分可属多个整体，生命周期可不同
- 组合: 部分只能属一个整体，生命周期相同

#### UML 14种图
类图、对象图、用例图、序列图、通信图、状态图、活动图、构件图、部署图、制品图、组合结构图、包图、交互概览图、计时图

#### UML 5种视图
用例视图（中心）、逻辑视图、进程视图、实现视图、部署视图

## 5. 多媒体技术

### 多媒体特征
1. **多维化**: 媒体多样化
2. **集成性**: 设备集成、信息集成
3. **交互性**: 有效控制和使用信息
4. **实时性**: 音频视频的时间特性

### 关键技术
- 视音频技术（数字化、编码、识别）
- 通信技术（信道、传输）
- 数据压缩技术（无损/压缩、即时/非即时）
- VR/AR技术（桌面式、分布式、沉浸式、增强式）
